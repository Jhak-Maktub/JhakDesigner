# Security Policy

## Reporting a Vulnerability

If you discover any security vulnerabilities, please report them by contacting us at [security@jhakmaktub.com](mailto:security@jhakmaktub.com). We take security seriously and will respond as quickly as possible to address your concerns.

## Supported Versions

We will actively support the current version and the last two major versions. You can find our release notes for each version in the [releases section](../../releases).
