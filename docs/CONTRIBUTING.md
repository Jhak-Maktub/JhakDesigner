# Contributing to JhakDesigner

Thank you for your interest in contributing to JhakDesigner! Here are some guidelines to help you get started:

## How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or bug fix.
3. Make your changes and commit them with a clear message.
4. Push your branch to your fork.
5. Submit a pull request.

## Code Style
- Follow the coding style used throughout the project.

## Issues
- Please refer to the existing issues to see if your bug or feature request is already being addressed.

Thank you for contributing!
